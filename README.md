# seanmusic

Website music shop 2hand design by Nhuong69.

## Cấu trúc

- `index.html`: khung trang và nội dung.
- `style.css`: toàn bộ phần giao diện.
- `script.js`: xử lý form liên hệ đơn giản.

## Chạy dự án

Chỉ cần mở trực tiếp `index.html` bằng trình duyệt.

Nếu muốn chạy qua server tĩnh, có thể dùng bất kỳ công cụ local server nào.
